So funktioniert es

Drei Schritte von der Auswahl bis zum Treffen. Keine versteckten Kosten, keine Umwege.

01

Entdecken

Filtern Sie nach Stadt, Sprache und Art der Begleitung. Jedes Profil zeigt Preise und Verfügbarkeit offen an.

02

Anfragen

Wählen Sie Datum und Dauer und senden Sie eine Anfrage. Eine kleine Anzahlung reserviert den Termin verbindlich.

03

Treffen

Sie erhalten eine Bestätigung. Den restlichen Betrag begleichen Sie direkt beim Treffen, ganz unkompliziert.

Das Versprechen

Diskret, verifiziert, klar.

Diskretion

Anfragen bleiben privat. Sie entscheiden, was Sie teilen und mit wem.

Verifizierung

Als verifiziert markierte Profile wurden persönlich geprüft.

Faire, klare Preise

Stundensätze stehen offen im Profil. Die Anzahlung wird voll auf den Gesamtpreis angerechnet, den Rest zahlen Sie erst beim Treffen.
